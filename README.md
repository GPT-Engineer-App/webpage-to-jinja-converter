# webpage-to-jinja-converter

An interface for the app: Webpage to Jinja-Template Converter - it takes a webpage and provides a UI for picking dom elements and giving them names, types, and defining template variables by selecting text, and finally saving/exporting them as jinja templates.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/webpage-to-jinja-converter.git
cd webpage-to-jinja-converter
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
